## Should you self-host?

**Use Cloud** if you want Asymptote running in minutes with zero infrastructure management.

**Use Self-Hosted** if you need data sovereignty, air-gapped environments, or custom LLM providers.

## Container Registry

Access to Asymptote's Docker images. [Contact Asymptote](mailto:founders@asymptotelabs.ai) for registry credentials.
