## Overview

This guide covers deploying Asymptote using Docker Compose on a single Linux VM. Choose between automated AWS provisioning with Terraform or manual setup for other environments.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Container fails to start">
    Check logs for errors:
    ```bash
    docker compose logs app
    ```
    
    Common issues:
    - Database connection string incorrect
    - Redis not reachable
    - Invalid license key
  </Accordion>
  <Accordion title="Database connection errors">
    Verify PostgreSQL is accessible:
    ```bash
    docker compose exec app pg_isready -h your-db-host
    ```
  </Accordion>
  <Accordion title="Out of memory">
    Increase VM memory or tune container limits in `docker-compose.yml`:
    ```yaml
    services:
      app:
        deploy:
          resources:
            limits:
              memory: 4G
    ```
  </Accordion>
</AccordionGroup>
