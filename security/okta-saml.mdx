# Okta SAML SSO

Configure SAML-based Single Sign-On with Okta for enhanced security and compliance.

## Prerequisites

- Okta administrator access
- Asymptote Enterprise plan
- Domain verification completed

## SAML Configuration

### 1. Create SAML Application

1. Log in to your Okta Admin Console
2. Navigate to **Applications** → **Applications**
3. Click **Create App Integration**
4. Select **SAML 2.0**
5. Click **Next**

### 2. Configure SAML Settings

#### General Settings

| Setting | Value |
|---------|-------|
| App name | Asymptote |
| App logo | Upload Asymptote logo (optional) |

#### SAML Settings

| Setting | Value |
|---------|-------|
| Single sign-on URL | `https://app.asymptotelabs.ai/auth/saml/callback` |
| Audience URI (SP Entity ID) | `https://app.asymptotelabs.ai` |
| Name ID format | EmailAddress |
| Application username | Email |

#### Attribute Statements

| Name | Value |
|------|-------|
| email | user.email |
| firstName | user.firstName |
| lastName | user.lastName |

### 3. Download Metadata

1. Go to the **Sign On** tab
2. Click **View Setup Instructions**
3. Download the **Identity Provider metadata** XML file

### 4. Configure Asymptote

1. Log in to Asymptote as an admin
2. Navigate to **Settings** → **Authentication** → **SAML**
3. Upload the metadata XML file
4. Configure the following:

```yaml
SAML Configuration:
  Entity ID: https://app.asymptotelabs.ai
  ACS URL: https://app.asymptotelabs.ai/auth/saml/callback
  Name ID Format: emailAddress
```

5. Click **Save**

### 5. Test Configuration

1. Click **Test SAML Login**
2. Authenticate with Okta
3. Verify successful authentication

## Advanced Configuration

### Just-In-Time Provisioning

Enable automatic user creation on first login:

```yaml
JIT Provisioning:
  Enabled: true
  Default Role: Developer
  Auto-assign Groups: true
```

### Group Mapping

Map Okta groups to Asymptote roles:

| Okta Group | Asymptote Role |
|------------|----------------|
| asymptote-admins | Admin |
| asymptote-developers | Developer |
| asymptote-viewers | Viewer |
