# Security Overview

Asymptote is built with security at its core. This document outlines our security practices and the features available to protect your organization.

## Data Security

### Encryption

- **In Transit**: All data is encrypted using TLS 1.3
- **At Rest**: Data is encrypted using AES-256

### Data Retention

- Scan results are retained for 90 days by default
- Configure custom retention policies as needed
- Delete data on demand

### Data Isolation

- Each organization's data is logically isolated
- No cross-tenant data access

## Access Control

### Role-Based Access Control (RBAC)

Define roles and permissions for your team:

| Role | Permissions |
|------|-------------|
| Admin | Full access |
| Developer | View and run scans |
| Viewer | View results only |
