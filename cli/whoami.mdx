# asym whoami

Display information about the currently authenticated account and session.

```bash
asym whoami
```

<Accordion title="Example output">
```bash
$ asym whoami

Email:         you@example.com
User ID:       73de6cbf-7f16-48d2-b869-19a6051927f7
Organization:  Acme Corp
Token:         ask_live_****
```
</Accordion>
