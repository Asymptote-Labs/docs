# asym version

Display the version number, git commit, and build date of the Asymptote CLI.

```bash
asym version
```

<Accordion title="Example output">
```bash
$ asym version

asym version 2.7.0 (1becf36) built on 2026-02-28T00:28:19Z
```
</Accordion>

## Flags

| Flag | Description |
|------|-------------|
| `-h, --help` | Help for version |
| `-v, --version` | Also available as `asym -v` at the top level |
