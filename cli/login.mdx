# asym login

Authenticate the CLI with your Asymptote account. Opens a browser window to complete authorization.

```bash
asym login
```

<Accordion title="Example output">
```bash
$ asym login

  Open the authorization page in your browser?  Yes

  Complete authorization at https://asymptotelabs.ai/cli/authorize/abc123

✓ Opened login window in browser

⡿  Waiting...

✓ Successfully authenticated as justin@acme.com
```
</Accordion>

Session data is stored in `~/.asymptote/config.yml`.

## Flags

| Flag | Description |
|------|-------------|
| `-h, --help` | Help for login |
