# asym upgrade

Update the embedded hooks binary for all platforms that have been previously installed (Claude Code, Cursor, Copilot, Factory).

```bash
asym upgrade
```

This detects which platforms have an `asym-hooks` binary installed and replaces each one with the version embedded in the current CLI build. Typically run automatically after `brew upgrade asymptote-cli`, but can also be run manually at any time.

To upgrade the CLI binary and hooks in one step:

```bash
brew upgrade asymptote-cli && asym upgrade
```
