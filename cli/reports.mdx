# Reports

Generate detailed security reports and view your scan history.

## Generate a Report

Create a security report for your project:

```bash
asymptote report [flags]
```

| Flag | Type | Description |
|------|------|-------------|
| `--format <format>` | string | Output format (html, pdf, json, markdown) |
| `--output <path>` | string | Output file path |
| `--period <period>` | string | Time period (7d, 30d, 90d) |
| `--include-fixed` | bool | Include resolved issues |

### Examples

Generate a PDF report:

```bash
asymptote report --format pdf --output ./security-report.pdf
```

<Accordion title="Example output">
```bash
$ asymptote report --format pdf --output ./security-report.pdf

Generating report...

Collecting scan data from last 30 days...
Analyzing 47 scans...

✓ Report generated: ./security-report.pdf

Summary:
 - Total scans: 47
 - Issues found: 23
 - Issues fixed: 19
 - Open issues: 4
```
</Accordion>

Generate a JSON report for CI/CD:

```bash
asymptote report --format json --output ./report.json
```

Generate a markdown report:

```bash
asymptote report --format markdown --output ./SECURITY.md
```

## View Scan History

View recent scans for your project:

```bash
asymptote history [flags]
```

| Flag | Type | Description |
|------|------|-------------|
| `--limit <n>` | number | Number of results to show (default: 10) |
| `--status <status>` | string | Filter by status (completed, failed, running) |

## View Scan Details

Get details about a specific scan:

```bash
asymptote history show <scan-id>
```

<Accordion title="Example output">
```bash
$ asymptote history show scan-abc123

Scan: scan-abc123
Date: 2025-01-15 14:32:15
Duration: 2.3s
Files scanned: 142
Status: completed

Issues (3):

 HIGH      hardcoded-secret        src/config.ts     23

 LOW       console-log             src/utils.ts      12

```
</Accordion>

## Export History

Export scan history to CSV:

```bash
asymptote history export --format csv --output ./history.csv
```
